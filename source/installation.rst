Installation
============
SHADERed can be installed easily, just follow the instructions for your platform

Debian/Ubuntu
*************

Install dependencies

.. code-block:: bash

    sudo apt install libopengl0 libsfml-audio2.5 libsfml-system2.5 libsfml-network2.5 libglew2.1 libsdl2-2.0-0 zlib1g

Then you can download the installation zip from https://shadered.org/, unzip it wherever you want, and then run the `SHADERed` binary.
You can alternatively do this using the command line:

.. code-block:: bash

    wget https://github.com/dfranx/SHADERed/releases/latest/download/Linux.zip
    unzip Linux.zip && rm Linux.zip
    ./SHADERed/SHADERed
